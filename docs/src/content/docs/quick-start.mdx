Make sure you have [.NET 10 SDK](https://dot.net/download), PostgreSQL, and Redis installed before you begin. See [Prerequisites](/docs/prerequisites/) for details.
</Aside>

## Get Started <Badge>3 steps</Badge>

<Steps>

### Clone the Repository

```bash
git clone https://github.com/fullstackhero/dotnet-starter-kit.git
cd dotnet-starter-kit
```

### Build the Solution

```bash
dotnet build src/FSH.Framework.slnx
```

### Run the API

<Tabs labels={["Standard", "With Aspire"]}>
<TabPanel>

```bash
dotnet run --project src/Playground/Playground.Api
```

The API will be available at `https://localhost:5001`.

</TabPanel>
<TabPanel>

```bash
dotnet run --project src/Playground/FSH.Playground.AppHost
```

This starts the API along with PostgreSQL, Redis, and the Aspire dashboard.

</TabPanel>
</Tabs>

</Steps>

<Aside type="note">
Visit `https://localhost:5001/scalar/v1` for the interactive API documentation powered by Scalar.
</Aside>

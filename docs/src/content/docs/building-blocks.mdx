Building Blocks are shared framework libraries consumed by all modules. They provide the foundational infrastructure that modules build upon.

:::caution
Changes to Building Blocks have wide blast radius. Modify with care тАФ all modules depend on these libraries.
:::

## Core

Base entities, domain events, CQRS interfaces, and specifications.

- `BaseEntity` тАФ `Id`, `CreatedAt`, `UpdatedAt`, `TenantId`
- `AggregateRoot` тАФ extends `BaseEntity` with domain events
- Marker interfaces: `IHasTenant`, `IAuditableEntity`, `ISoftDeletable`

## Persistence

EF Core base `DbContext`, repository pattern, and migration infrastructure.

## Web

Minimal API extensions, middleware, global exception handling (`ProblemDetails`), and module loading.

## Caching

Redis-backed distributed caching service abstraction.

## Eventing

Integration event infrastructure for cross-module communication.

## Authorization

Permission-based authorization extensions for minimal API endpoints.

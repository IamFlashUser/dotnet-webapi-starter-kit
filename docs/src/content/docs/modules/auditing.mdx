The Auditing module provides automatic audit trail for entity changes.

## How It Works

Entity changes are captured automatically through EF Core interceptors. When entities implementing `IAuditableEntity` are created, updated, or deleted, an audit record is stored.

## Audit Record

Each audit entry captures:

- **User** — who made the change
- **Entity** — what was changed
- **Action** — create, update, or delete
- **Timestamp** — when it happened
- **Old/New values** — what changed
- **Tenant** — tenant context

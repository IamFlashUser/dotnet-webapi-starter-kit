Before you begin, ensure you have the following installed:

## Required

- [**.NET 10 SDK**](https://dot.net/download) (or later)
- [**PostgreSQL**](https://www.postgresql.org/download/) (16+ recommended)
- [**Redis**](https://redis.io/download/) (for caching)

## Recommended

- [**.NET Aspire workload**](https://learn.microsoft.com/en-us/dotnet/aspire/) — for local development orchestration
  ```bash
  dotnet workload install aspire
  ```
- **Docker** — for running PostgreSQL and Redis via containers
- **IDE** — Visual Studio 2022, VS Code with C# Dev Kit, or JetBrains Rider

## Verify Installation

```bash
# Check .NET SDK
dotnet --version

# Check Aspire workload
dotnet workload list
```

---
title: Adding a Module
description: How to create a new bounded context module in fullstackhero.
---

## 1. Create the Projects

Under `src/Modules/{Name}/`, create two projects:

```
Modules.{Name}/              # Runtime implementation
Modules.{Name}.Contracts/    # Public API
```

## 2. Implement IModule

```csharp
[FshModule(Order = 4)]
public class CatalogModule : IModule
{
    public void ConfigureServices(IHostApplicationBuilder builder)
    {
        // Register DbContext, services, etc.
    }

    public void MapEndpoints(IEndpointRouteBuilder endpoints)
    {
        var group = endpoints.MapGroup("catalog").WithTags("Catalog");
        // Map feature endpoints
    }
}
```

## 3. Add DbContext

Create a module-specific `DbContext` extending the framework base.

## 4. Register the Module

Add the module assembly to the `Program.cs` module assemblies array.

## 5. Add Migrations

Create a migration project if your module has a database context.

## 6. Add Tests

Create `Tests/{Name}.Tests/` and add architecture test rules to enforce module boundaries.
